## 题目九: 斐波那契数列
写一个函数，输入n，求斐波那契（Fibonacci）数列的第n项。

```java

//这里提一个效率很低的解法:

class Test{
    long Fibonacci(int n) {
        if(n <=0 ) {
            return 0; 
        }
        if(n==1) {
            return 1;
        }
        
        return Fibonacci(n-1) + Fibonacci(n-2);
    }   
}
```

> 尝试O(n)的解法
