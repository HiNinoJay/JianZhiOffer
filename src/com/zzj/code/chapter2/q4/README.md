## 题目四 替换空格
请实现一个函数，把字符串汇总的每个空格替换成"%20"。
例如，输入"We are happy"
则输出"We%20are%20happy"

假设面试官让我们在原来的字符串上进行替换，
并且保证输入的字符串后面有足够多的空余内存。


> tips: URL空格替换，空格的ASCII为32，即十六进制的0x23
>
> 时间复杂度为O(n2)的解法，不足以拿到Offer
>
> 寻找时间复杂度为O(n)的解法
>
>
>
